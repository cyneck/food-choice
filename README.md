# 🍽️ 食分choice - 每日三餐智能推荐

一个简单的 Web 应用，帮助你解决"今天吃什么"的纠结问题。

## ✨ 功能特性

- **智能推荐**：为早、中、晚餐分别推荐餐食
- **多种筛选**：支持快手菜、健康、素食、重口味等模式
- **防重复**：避免连续推荐相同餐食
- **收藏系统**：收藏喜欢的餐食
- **响应式设计**：完美适配手机和电脑

## 🚀 使用方法

### 本地运行

1. 直接双击 `index.html` 打开
2. 或使用命令行：
   ```bash
   npx http-server -p 8080
   ```

### GitHub Pages 部署

应用已配置为 GitHub Pages 静态站点，可直接访问。

## 🎨 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
- LocalStorage
